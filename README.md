## Getting Started
### Prerequisites
- Node.js v22+
- npm v10+
### Installation
1. Clone repository:
```git clone https://github.com/OneWorldQuest/OWQ-Website.git```
3. Navigate to project directory
```cd OWQ-Website.git```
5. Install dependencies
```npm install```
4. Start dev server
```npm run dev```
the site should now be running at http://localhost:5173
